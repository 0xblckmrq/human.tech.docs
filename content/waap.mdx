# WaaP (Wallet as a Protocol)

[Visit the WaaP home page](https://human.tech/waap)

WaaP is a blockchain wallet solution that combines user convenience with security. The platform leverages Two-Party Computation (2PC) architecture — no seed phrases, no custodial risk — to protect user assets.

## Key Features

The wallet supports streamlined onboarding through multiple methods like email, phone, and biometric authentication. It implements a split private key security model where private keys are split across two parties, with no single point of compromise.

WaaP operates across multiple blockchains including Ethereum and Sui, and provides a Gas Tank feature allowing developers to sponsor user transactions. Notably, it functions as a protocol rather than a hosted service, eliminating recurring subscription fees.

The platform emphasizes trustless account recovery, enabling users to regain access independently without relying on external custodians.
