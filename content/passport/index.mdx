# Human Passport Overview

[Visit the Human Passport home page](https://passport.human.tech/)

Human Passport is an identity verification platform designed to combat fraud in blockchain ecosystems. Over 2 million Passports created, protecting more than $500 million in airdrop and grant funds.

## Core Offerings

The platform provides three main products:

1. **Stamps** — Verifiable credentials allowing users to demonstrate trustworthiness through connected accounts and attribute verification, resulting in a Unique Humanity Score while maintaining privacy.

2. **Models** — Machine learning-based scoring that analyzes EVM addresses using on-chain activity, achieving 96% human detection accuracy without requiring user participation.

3. **Data Services** — Custom analysis for airdrops and grants, featuring clustering analysis and expert review capabilities.

## Zero-Knowledge Technology

Passport incorporates a privacy-focused protocol powered by Human Keys that enables users to verify personhood, residency, and government identification without exposing sensitive information. The system offers configurable privacy controls for various regulatory environments.
