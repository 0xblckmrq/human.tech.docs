# X Stamp (Retired)

> **This stamp has been retired** and is no longer available for verification.

The X Stamp (formerly Twitter Stamp) has been **retired** due to changes in the Twitter/X platform that made reliable verification no longer feasible.

## Status

| Detail | Value |
| --- | --- |
| **Status** | RETIRED |
| **Reason** | Twitter/X platform changes |

## Previous Requirements

When the X Stamp was active, it required:

- A verified X (Twitter) profile
- 100 or more followers
- An account at least 1 year old

## Why Was It Retired?

Changes to the Twitter/X platform, including API access restrictions and modifications to the verification system, made it impossible to maintain reliable and consistent verification. As a result, the stamp was retired from the Passport ecosystem.

## Impact on Existing Stamps

If you previously held the X Stamp, it is no longer active on your Passport and does not contribute to your Unique Humanity Score.
